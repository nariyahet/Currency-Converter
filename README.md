# 💱 Currency Converter

A modern and responsive **Currency Converter Web Application** built using **HTML, CSS, and JavaScript**.

This application allows users to convert currencies using real-time exchange rates with an interactive and user-friendly interface.

## 📌 About The Project

Currency Converter is a frontend web application that helps users convert one currency into another quickly and easily.

The project uses an exchange rate API to fetch currency values and calculate the converted amount instantly.

This project demonstrates:

- JavaScript API Integration
- Fetch API & Async/Await
- DOM Manipulation
- Dynamic Data Handling
- Responsive Web Design

## 🎯 How To Use

1. Enter the amount you want to convert.
2. Select the currency you want to convert from.
3. Select the currency you want to convert into.
4. Click on the **Get Exchange Rate** button.
5. View the converted currency value.
